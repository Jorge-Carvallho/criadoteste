Arrays em JavaScript: Conceitos e Aplicações

O que são Arrays?

Um array é uma estrutura de dados que armazena múltiplos valores em uma única variável. Os valores são armazenados em posições chamadas índices, começando do índice 0.

Como Criar um Array

let frutas = ["maçã", "banana", "laranja"];

Aqui, criamos um array chamado frutas contendo três elementos: "maçã", "banana" e "laranja".

Como Acessar Elementos

console.log(frutas[0]); // saída: maçã
console.log(frutas[2]); // saída: laranja

Acessamos os elementos de um array usando os índices. O primeiro elemento tem índice 0.

Modificar um Array

frutas.push("uva"); // adiciona 'uva' ao final
frutas.pop(); // remove o último elemento ('uva')

push: Adiciona um elemento ao final do array.
pop: Remove o último elemento.

Iterar sobre Arrays

for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

Usamos um loop para iterar sobre os elementos de um array e realizar operações em cada item.

Métodos Úteis

length: Retorna o tamanho do array.
shift(): Remove o primeiro elemento.
unshift(): Adiciona um elemento no início do array.
indexOf(): Retorna o índice de um elemento específico.

Arrays em Ação

Arrays são fundamentais no desenvolvimento web, permitindo organizar e manipular grandes conjuntos de dados. Combinados com loops e métodos de array, eles oferecem flexibilidade para solucionar uma ampla gama de problemas em JavaScript.
